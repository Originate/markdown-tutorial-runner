### verify that a file has the given content

Verifies that a file with the given name exists,
and has the given content.


#### Example

<a class="tutorialRunner_runMarkdownInTutrun">
```markdown
<a class="tutorialRunner_verifyFileContent">

__hello.txt__

`​``
hello world
`​``
</a>
```
</a>
